# 基于OpenCV模板匹配和PySide6的数字识别

#### 介绍
一个菜鸡写的py大作业，用pyside6搭建界面，opencv模板匹配完成识别，字模是电赛送药小车的。

#### 软件架构
界面：PySide6

图像处理/识别：OpenCV


#### 安装教程

Windows下激活.venv中的虚拟环境然后运行main.py即可。

Linux因为没有requirements.txt，所以得手动装一下依赖。

#### 使用说明

1.  Python版本 3.11
2.  模板识别有点捞，角度和光照影响挺大的
